# Constatos - Back e Front
Repositório de Teste de API + Front.

## Sobre:
Este repositório contém duas pastas iniciais, uma para um back-end feito em JavaScript + Node.js e ou para um front-end feito em React. O propósito desta simulação é apenas de treinar e exemplificar conceitos e foi pensado para quem se interessar poder baixar o repositório inteiro e testar sem a necessidade de instalação de um SGBD, oque facilita os testes.

## Sobre o Back-End:
O back-end consite de uma API construída em JavaScript + Node.js que vai simular um banco de dados através do uso de uma coleção de objetos, o que claro, não vai permitir em principio a persistência de dados já que não vamos usar um SGBD, ou seja, desligando o API todos os contatos cadastrados serão perdidos "reiniciando" nossa simulação de banco de dados do zero.

## Sobre o Front-End:
O nosso front-end será uma SPA construida em React, será simples, feita para exercitar conceitos gerais do React, em especial a integração com uma API e uso de hooks.

## Observação
Não é meu intuito cobrir tudo sobre os assuntos citados acima, apenas treinar sua implementação e trabalhar técnicas de programação, tudo com comentários.
